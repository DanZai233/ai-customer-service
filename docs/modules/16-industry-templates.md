# 行业模板

## 职责

为常见客服行业提供可查看、可复制、可安装的场景化启动资料，减少从空白知识库开始整理的成本。

## 页面与接口

- 页面：`/templates`
- 列表：`GET /api/templates`
- 安装：`POST /api/templates/{templateId}/install`

列表要求 `knowledge.read`，安装要求 `knowledge.write`。

## 内置行业

当前包含 8 套模板、24 篇知识草稿：

1. 电商订单与售后
2. SaaS 试用与订阅支持
3. 教育课程咨询与教务服务
4. 医疗预约与就诊服务
5. 金融账户与交易服务
6. 旅游预订与行程保障
7. 本地生活履约与客诉
8. 制造业询报价与技术售后

每套模板包含适用团队、典型客户问题、开放 API JSON 案例、知识文章和明确的转人工边界。

## 安装规则

模板文章以 `template://{templateId}/{documentSlug}` 标记来源。数据库对组织和来源地址建立唯一索引，因此重复安装不会产生重复文章；缺失文章可通过“补齐”恢复。

安装后统一进入草稿状态。管理员需根据本企业真实合同、价格、时效和政策审核，再到知识库发布。医疗、金融等模板明确禁止 AI 做诊断、收益承诺或绕过合规核验。

## 验收

1. 搜索和行业筛选能缩小模板范围。
2. 模板详情能展示全部文章与 API 示例。
3. 点击安装后知识库新增对应草稿。
4. 再次安装返回零新增，文章数量不变。
5. 已安装模板可跳转到知识库继续编辑和发布。
