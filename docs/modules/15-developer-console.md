# 开发者中心

## 职责

让管理员无需修改部署配置即可创建、查看和吊销开放 API 密钥，并复制可直接改造的接入示例。

## 页面

- `/developer`

页面受 `settings.read` 权限保护；创建和吊销密钥要求 `settings.manage`。

## 功能

1. 显示有效密钥、接口数量和 OpenAPI 版本。
2. 创建带权限范围和有效期的密钥。
3. 新密钥仅显示一次，支持一键复制。
4. 提供 cURL、JavaScript、Python 和 Next.js 服务端代理示例。
5. 展示接口清单并可直接打开 `/api/openapi.json`。
6. 显示密钥最后使用时间、到期时间和吊销状态。

## 安全约束

- 接入示例只面向服务端调用。
- 浏览器和移动端不得内置 `luma_live_` 密钥，应调用自有服务端代理。
- 推荐按生产、测试和具体集成分别创建密钥，并只授予必要权限。

## 验收

1. 管理员创建密钥后，只在当前页面看到一次完整值。
2. 复制示例调用公共 API 后，最后使用时间发生更新。
3. 吊销密钥后，公共 API 返回 `401 invalid_api_key`。
